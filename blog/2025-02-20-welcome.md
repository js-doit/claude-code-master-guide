**Claude Code 마스터 가이드**에 오신 것을 환영합니다.

이 가이드는 Claude Code를 처음 접하는 개발자부터, 기업 AI 팀을 구축하려는 엔지니어까지 — 모든 수준의 개발자를 위해 만들어졌습니다.

<!-- truncate -->

## 왜 이 가이드인가?

Claude Code는 강력한 도구지만 공식 문서의 대부분이 영어입니다. 한국어로 된 체계적인 학습 자료가 없다는 아쉬움에서 이 프로젝트를 시작했습니다.

## 무엇을 배울 수 있나요?

5단계 커리큘럼을 통해 다음 내용을 다룹니다:

- **레벨 1 — 입문**: 설치, API 키 설정, 첫 번째 실행
- **레벨 2 — 기초**: CLAUDE.md, 슬래시 커맨드, Git 연동, 메모리 시스템
- **레벨 3 — 중급**: Hooks, MCP 서버, 비용 최적화, CI/CD 통합
- **레벨 4 — 고급**: 커스텀 스킬, Agent SDK, 대규모 코드베이스 전략
- **레벨 5 — 마스터**: 멀티에이전트 시스템, 기업 AI 팀 구축

## 앞으로 이 공간은?

Claude Code는 Anthropic이 활발하게 개발 중인 툴입니다. 새로운 기능이 출시될 때마다 이곳에 릴리즈 노트와 활용 팁을 공유하겠습니다.
