# 🏆 캡스톤 — 기업 팀 에이전트 구축

이 캡스톤 프로젝트를 완료하면 CEO 오케스트레이터를 중심으로 마케팅, HR, 개발, 재무, 영업, 법무, CS 팀 에이전트로 구성된 완전한 기업 AI 팀을 구축할 수 있습니다.

## 프로젝트 목표
- 멀티에이전트 기업 시스템의 전체 아키텍처를 설계한다
- 각 부서별 전문 에이전트를 개발하고 통합한다
- CEO 오케스트레이터로 전사 워크플로우를 조율한다
- 업종별 템플릿으로 즉시 적용 가능한 솔루션을 제공한다
