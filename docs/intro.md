# 👋 시작하기

> Claude Code 마스터 가이드에 오신 것을 환영합니다.

이 가이드는 Claude Code를 처음 접하는 입문자부터 기업 AI 팀을 구축하는 마스터까지, 단계별로 완전 정복할 수 있도록 설계되었습니다.

## 나에게 맞는 경로 선택하기

### 🚀 입문자라면
설치부터 차근차근 시작하세요.
→ [레벨 1 — 입문 시작하기](/docs/level-1/intro)

### 🔧 개발자라면
설치는 건너뛰고 실전 활용법부터 시작하세요.
→ [레벨 2 — 기초 시작하기](/docs/level-2/intro)

### 🏢 기업 도입을 검토 중이라면
멀티에이전트 아키텍처와 ROI 분석부터 확인하세요.
→ [레벨 5 — 마스터 시작하기](/docs/level-5/intro)
